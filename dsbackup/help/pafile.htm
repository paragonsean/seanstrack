Part Results File:

This specifies how to name the zip file that will contain the shot and parameter data.

This file will always have a .PAZ extension. You specify the name of the file by combining the static parts of the name with tags that the program will replace with dynamic content.

The tags that can be used for this file are listed below.
TagContent
{c} Computer
{m} Machine
{p} Part
{dd} Day of the month [1,31]
{mm} Month [1,12]
{yyyy} Year [2002]
{yy} Year [02]
{hh} Hour [0,23]
{mi} Minutes [0-59]
{ss} Seconds [0-59]
{s} Shift Number
{nb} First Shot Number
{ne} Last Shot Number

As an example, for machine M01 and part VISITRAK the program would have originally saved the file containing shots 1000 to 2000 as follows 

C:\V5BACKUP\M01_VISITRAK_1000_2000.PAZ
To make the new name the same as the old one you would use the following. 
C:\V5BACKUP\{m}_{p}_{nb}_{ne}
if you would like to put the results for each day in a separate folder, you could do the following. 
C:\V5BACKUP\{mm}_{dd}\{m}\Shots_{nb}_to_{ne}
Which would create a new folder if necessary and would save the above file as 
C:\V5BACKUP\03_20\M01\Shots_1000_to_2000.PAZ
