Sample behavioral waveforms for design file clk_divider.vhd

The following waveforms show the behavior of lpm_counter megafunction for the chosen set of parameters in design clk_divider.vhd. The design clk_divider.vhd is a 10 bit up counter. The counter has a synchronous clear input.

Fig. 1 : Wave showing counter operation.

The output port cout will be asserted at the completion of count sequence. The ports cin and cout are used to chain multiple counters together.
