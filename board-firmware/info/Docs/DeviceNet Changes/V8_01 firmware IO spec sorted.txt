FT2 I/O structure, V8.01 and later firmware
Must use control_all.txt 7-14-10 or later

ISW1, Existing Opto 22 based inputs.
Use command OI1<CR> to read.

ISW2, All 32 bits are dedicated for monitor inputs
OI2<CR> to read

ISW3, All 32 bits are dedicated for control inputs
OI3<CR> to read.

OSW1, Existing Opto 22 based outputs. FT2 I/O numbers are the same as TrueTrak Wire Numbers.
OO1<CR> to read
OO1=Hxxxxxxxx to set individual bits
OO1=~Hxxxxxxxx to clear individual bits

OSW2, All 32 bits are dedicated to monitor outputs. TrueTrak wire numbers should match FT2 I/O numbers.
OO2<CR> to read
OO2=Hxxxxxxxx to set individual bits
OO2=~Hxxxxxxxx to clear individual bits

OSW3, All 32 bits are dedicated to control outputs. TrueTrak wire numbers should match FT2 I/O numbers.
OO3<CR> to read
OO3=Hxxxxxxxx to set individual bits
OO3=~Hxxxxxxxx to clear individual bits
