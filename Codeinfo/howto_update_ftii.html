1. Pull down the "Help" menu of the Plot program and choose "FasTrak2 Help".
  2. Under "Misc", click on "External Interface Boards".
  3. Look at your external interface board and compare it to the photos to determine which you have (Rev B or Rev E).
  4. If you don't have a Rev E board give me a call.
  5. Shut down all True-Trak software.
  6. Rename the current C:\Ftii directory to C:\Ftii_Old
  7. Copy the ftii_files.exe program to the root directory and run it from there. It will create a C:\Ftii directory and copy the new ftii files there.
  8. Unplug any network cables to the monitoring computer. You should only have one cable from the cpu board to the blue ethernet switch and another from the blue ethernet switch to the FasTrak2 board.
  9. Run the "terminal.exe" program in the C:\Ftii directory.
  10. The title of this program is "FasTrak2 Terminal" followed by the ip address of the FasTrak2 board. If this is not the address of the board that you wish to update then pull down the "Edit" menu and choose "Address". Enter the ip address of the board in the editbox to the right of "Tcp/Ip Address". The "Port" should be "20000". Press the "Save Changes" button for this to take effect.
  11. At the bottom of the screen is an editbox. Enter "OV" ( letter o letter v) and press Enter. This will ask the board to identify the version of firmware that is currently loaded.
    You should get a response from the board similar to the following. 
    R:5#cyc_chip_14010201
R:6#cyc_7_04, monitor_and_control, interrupt driven control, polled Ethernet, UDP, handshake, ad_loop19
    If the response begins with "R:6#cyc" then this is a newer version of the FasTrak2 board called a "Cyclone" board. If the response begins with "R:6:emc" then this is the older FasTrak2 board.
    The number after the above strings is the current firmware version. In the example above the firmware version is 7_04. On yours it will say 4_14c. I am pretty sure you have one of the older FasTrak2 boards and not the Cyclone board.
  12. Pull down the "File" menu and choose "Upload File".
  13. Press the "Flash 6" radio button if this is the older FasTrak2 board or "Flash 7" if this is a Cyclone board.
  14. Make sure the "All Files" box is checked.
  15. Press "Begin" to upload all the files.
  16. When the uploads are complete, enter "NM" in the editbox at the bottom of the screen and press Enter. The board should respond with the current ethernet settings. 
    # Network Settings

# N1	ethernet address: 14:13:12:11:16:15
# N2	ip address: 192.168.254.99
# N3	nameserver ip address: 165.227.1.1
# N4	subnet mask: 255.255.255.0
# N5	gateway ip address: 10.0.0.255
# N6	Reserved
#
# N7	Remote IO ip address: 10.0.0.55
# N8	Remote IO port: 502
# N9	Remote IO type: 1
#	1 == MODBUS
# NM	To view this menu
# NF	To flash network settings after making changes
  17. Even though these appear correct, reenter the tcp/ip address of the board.
    N2 192.168.254.99
    and press Enter.
    Be sure to enter a space after the "N2".
  18. Flash this address by entering "NF" and pressing Enter.
  19. Reset the FasTrak2 board.
  20. Start the True-Trak software.
