How To Set Up a Monitoring Computer on a Network

The monitoring computer is defined as the computer that is connected to the FasTrak or FasTrak2 board and collects the data at the end of each shot. The following assumes you are setting up a network of only the monitoring computers, office workstations, and data archiver and that the monitoring computer is running Windows 98.

Computer Name and Workgroup Name

  • Right click on the Network Neighborhood and select Properties.
  • Click on the Identification tab.
  • The Computer name should be set to a three character name such as W01, W02, etc. If you are running True-Trak Version 6.11 or higher the three character limit does not apply.
  • The Workgroup should be set to VISITRAK.

Network Setup

  • Click on the Configuration tab.
  • You should see the following.
    • Client for Microsoft Networks
    • A network adapter of some sort.
    • Either NetBEUI protocol or TCP/IP protocol.
      • If you have the NetBEUI protocol there is nothing to configure.
      • if you have TCP/IP protocol you should.
        • Click once on the icon next to the TCP/IP. This will highlight that line.
        • Click on the Properties button to bring up the TCP/IP Properties dialog.
        • Set the IP Address to 192.168.254.101. The last of these four numbers, 101, must be different for each computer. We normally start with 101 and then use 102, 103, etc.
        • Set the Subnet Mask to 255.255.255.0.
        • Press OK to close the dialog.
    • File and printer sharing for Microsoft Networks. If this is not present then...
      • Click on the File and Print Sharing... button.
      • Check the I want to give others access to my files checkbox.
      • Press the OK button to close the dialog.
  • Remove any other protocols and clients.
  • Press OK.
  • After you reboot, log in with the same name as the computer name and No password.

Setting the Computer to Boot Directly Into Windows

  • Right click on the Network Neighborhood and choose Properties.
  • In the listbox under Primary Network Login choose Windows Login
  • Press the OK button.
  • Windows will ask if you want to reboot. Click on the No button.

Sharing the V5 Directory

  • Run "My Computer"
  • Right click the C:\V5 folder.
  • Choose "Sharing..."
  • Click the Shared As button and the Full Button
  • Click "OK"

Connecting the Computers

The main office workstation, usually called WS0, should have the True-Trak Network Setup program installed. Run this program. The following assumes you have Windows XP installed on the office workstation. If you don't have Windows XP the process is the same but the details are a little different. The idea is to access the computers through the Network Neighborhood and never as local drives.

If the monitoring computer you are installing already appears in the list of computers displayed by the program, usually because you are installing a new hard drive, do the following.

  • Click once on the existing line for the monitoring computer you are installing. This will highlight the line.
  • Press the Remove Computer(s) button.
  • Press the OK button when assed if you wish to continue.
Add the new computer as follows.
  • Press the Brouse For Computers button.
  • Press the '+' next to My Network Places.
  • Press the '+' next to Entire Network.
  • Press the '+' next to Microsoft Windows Network.
  • Press the '+' next to VISITRAK.
  • Press the '+' next to the Monitoring Computer you are adding..
  • A V5 folder will appear. Click once on this folder to add it to the list of computers.
  • Press the OK button to close the dialog.
  • Click once on the new line, the one that represents the new monitoring computer, that has been added to the listbox.
  • Make sure the Type is set to Monitoring Computer.
  • Make sure there is a Yes in the Shared Full column on the right side of the screen.
  • Highlight all the computers in the list by clicking once on the first entry and then holding down the Shift key while clicking on the last entry.
  • Press the Standard Connections button to connect all of the computers. This will only affect the new monitoring computer, the office workstation(s), and the data archiver.
  • Press Save Changes to save the changes to each computer.
Note that the changes will not take place until the True-Trak programs have been restarted on the computers that have changed.
