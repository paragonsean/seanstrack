How To Restore Files on the Netware Server Using ArcserveIT

  • Turn on the P166 computer next to the server.
  • Log in as ADMIN/XMAS
  • Double click the Arcserve icon. This runs the f:\arcserve.6\manager program
  • You will be on the Source tab. There is a dropdown combobox at the top of this tab. Choose Media View from this list.
  • Open the tape you want by clicking on the little blue sideways triangle to the left of the tape name. Then choose the VISITRAK_SYS as your starting point and find the file you want.
  • Click the green box, it will fill in, to the left of any file you want restored.
  • Click the Required Media Tab. It should show the tape you want.
  • Enter the password XMAS using all caps.
  • Retype the password. Note that I can conceive of no reason why you should have to retype it. Just do it.
  • Press the Apply button.
  • Press the run button above the tabs. This button is not labeled but looks like a green stoplight.
  • Watch the Arcserve message screen on the server to follow the progress of your restore.
