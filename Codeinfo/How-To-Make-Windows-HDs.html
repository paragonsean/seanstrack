How to Make Windows Hard Drives (Win95, Win98) for Advantec and Ampro Boards

  • The following assumes you are using a 60GB HD. The procedure is a little different depending upon the size. The goal is to make one or two 20GB partitions on the drive.
  • There is a drive on the cpu already. This is Win95 and is configured as the master. Put the new, unpartitioned, drive on as the slave and boot into Win95.
  • Open a Dos window.
  • Enter FDISK at the command line. When it asks if you want to enable large disk support answer Yes.
  • Change disk, option 5, to the second disk (60GB).
  • Create a primary dos partition. When it asks if you want to use the maximum size, answer No.
  • Set the size to 33%
  • Exit.
  • Exit the dos window (type EXIT).
  • Restart Windows.
  • Open a DOS window.
  • Enter the following:
    FORMAT   /U   D:
    D:
    DIR  C:\*.ZIP   <<< This will let you see the names of the zip files.
    PKZIP25  -EXT  -DIR  -ATT=ALL  C:\WC95.ZIP
  • Exit to Windows
  • Copy either the V575.exe or V6.17.exe to the D drive
  • Shutdown
  • Put the new drive in as the Master.
  • Boot from the floppy
  • Run FDISK
  • Set partition 1 to be the ACTIVE partition.
  • Remove the floppy disk and reboot.
