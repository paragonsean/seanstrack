How to Install Maximizer Enterprise

  • Insert the Maximizer Enterprize CD into the drive
  • Choose Install Maximizer Enterprise
  • Enter serial number and purchase date (Aug 2000)
  • Choose Custom install
  • On the components screen choose Program Files and press the Change... button
  • Uncheck MaxFinder and MaxAlarm
  • Uncheck the Create desktop shortcut for ListWarehouse
  • Under the BTrieve Database Engine Setup choose Multiple computers on a BTrieve Client/Server Network
  • When the installation is complete, completely remove the Pervasive Client Software and reinstall it.
  • Run Maximizer Enterprise and close the sample addressbook.
  • Open a New Address Book with Name=Visi-Trak - Server and Location=F:\Data\Maximizer
