1. On the Ampro computer in Jay's room put the XP Embedded Ampro 700 Master disk as the master and the new disk as the slave.
  2. Use Maxblast 3 to set the new disk up as a boot disk, copying all the files from the Master disk.
  3. When the setup is done Max Blast will shut down the computer. Remove the master disk and put the new disk on as the master.
  4. When the new disk wakes up Max Blast will run automatically. Close it.
  5. Remove Max Blast 3 (Control Pannel, Add/Remove Programs)
  6. Run fbreseal.exe
  7. Reboot
  8. Log in as Administrator with no password
  9. Run Login Setup
  10. Uncheck "Users must enter..."
  11. Ok
  12. Select Administrator as the user to log in automatically
  13. Delete the Login Setup icon.
  14. Right click My Computer and select Properties
  15. Click on the Computer Name tab.
  16. Click on the Change... button.
  17. Replace the computer name with W01 and press OK.
  18. Press OK, OK and then Yes to reboot. This reboot will delete the fbreseal.exe from the desktop.
  19. Control Panel, Network Connections, Local Connection, TCP/IP, Properties
  20. Click the Use the following IP Settings
  21. IP Address 192.168.254.100
  22. Subnet Mask 255.255.255.0 (this will be filled in automatically when you click on the editbox).
  23. Click OK and then Close the window.
  24. Start, Programs, Internet Explorer
  25. Tools, Internet Options
  26. Home Page, Click on Use Blank
  27. Click the Advanced tab
  28. Uncheck Install on Demand (Other)
  29. Uncheck Enable offline items to be syncronized on a schedule
  30. Uncheck Use Smooth Scrolling
  31. Press OK
