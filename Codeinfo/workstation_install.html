Office Workstation Installation Steps

The following assumes that you are installing the Office Workstation for the first time.
  • Go to http://www.visi-trakworldwide.com/get.htm and get the v656.exe file.
  • Copy this to the root directory of the C drive on the Office Workstation and run it to create a C:\Install6.56 directory. If that directory already exists, just tell the program to replace always.
  • Run the Setup.exe program in that directory
  • Press the Install New TrueTrak System button. A screen with options will appear.
  • At the bottom of the screen check the checkbox for This is an Office Workstation or Data Archiver.
  • If you have any Sure-Trak™ Control Systems check the SureTrak Control Installed checkbox.
  • Press the Continue button to install the software
  • You will have created a new C:\V5 directory. Make this Shared Full
    • Right click on the V5 directory
    • From the popup menu choose Sharing and Security...
    • Check Share this folder on the network
    • Check Allow network users to change my files
  • Go to each of your monitoring computers and make sure that the V5 directory is shared full.
  • Go to http://www.visi-trakworldwide.com/getnet.htm and get the ns.exe file.
  • Copy this to the root directory of the Office Workstation.
  • Run the ns.exe from the root directory to create the C:\Install.ns directory on the Office Workstation.
  • Run the C:\Install.ns\Setup.exe program.
  • Press the Begin button to install the software.
  • Press the Esc key to exit from the install program when it is finished.
  • Make sure all the TrueTrak computers are online.
  • You should have a new icon on the desktop for the NetSetup program. Double click this to run the NetSetup program
  • As this is the first time you have run this program the Browse For Folder dialog box will automatically appear, prompting you to and browse to the V5 directory.
  • Do the following for each TrueTrak computer on the network, including the Office Workstation.
    • Traverse the network as instructed in the dialog box, starting at the Network Neighborhood.
    • When the folder icon for the V5 directory is visible, click on it ONCE and it will be added to the listbox of the main program.
  • Look at the line in the main listbox for each computer that you added. The Type should be Monitoring Computer for all computers except the Office Workstation you are installing. The Office Workstation should be shown as an Office Workstation. Change the Type of any that are not correct.
  • Select all the computers in the main listbox (click the first and then shift-click the last).
  • Press the Standard Connections button
  • Press the Save Changes button
  • Run the True-Trak 2020(tm) program on the Office Workstation by double clicking the new icon on the desktop. You should be able to see shots from all the monitoring computers.
