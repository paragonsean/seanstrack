Data Archiver Installation Steps

  • Go to http://www.visi-trakworldwide.com/getv7.htm and get the v706.exe file.
  • Copy this to the root directory of the C drive on the Data Archiver and run it to create a C:\Install7.06 directory. If that directory already exists, just tell the program to replace always.
  • Run the Setup.exe program in that directory
  • Press the Install New TrueTrak System button. A screen with options will appear.
  • Change the directory from C:\V5\ to C:\V5DS\
  • At the center of the screen set the computer type to Data Archiver.
  • Press the Continue button to install the software
  • You will have created a new C:\V5DS directory. Share this directory with full access.
  • Run the C:\Install7.06\Setup.exe program again.
  • Press the Install New TrueTrak System button.
  • At the center of the screen set the computer type to Local Installation on the Data Archiver.
  • Press the Continue button to install the local copy of the software.
  • Go to http://www.visi-trakworldwide.com/getda7.htm and get the da7.exe file.
  • Copy this file to the root directory of the Data Archiver and run it from there to create the C:\Install.da7 directory.
  • Run the C:\Install.da7\Setup.exe program
  • Press the Begin button to install the software.
  • Press the Esc key to exit from the install program when it is finished.
  • You should have a new icon on the desktop for the Data Archiver Backup Program. Double click this to run the program.
  • Pull down the Edit menu and choose Global File Settings and then press the Save Changes button and then close the program.
  • Make sure all the TrueTrak computers are online.
  • Run your NetSetup program. You should already have this installed. You should only have one installation as the program saves the configuration file in it's local directory.
  • If there is a current entry for the Data Archiver, remove it
  • Press the Browse for Computers button and browse to the new Data Archiver and click once on the V5DS directory to add it to the list of computers.
  • Select all the computers in the main listbox (click the first and then shift-click the last).
  • Press the Standard Connections button
  • Press the Save Changes button
