How to Make Install Disks

EasyTrak DOS

  1. Turn on the MS-DOS (Percolator) computer
  2. FORMAT /S /U A:
  3. CD \SNAPTRAK
  4. COPY *.* A:

TrueTrak V5.50h Install/Update

  1. Format 3 floppy disks
  2. Run Setup.exe in the D:\Install\Install5.64 folder.
  3. Press the Create Installation Floppies button
  4. Follow the directions
