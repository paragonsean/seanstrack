Computer Naming Conventions

WS0(W S zero) The Office Workstation. This computer displays the data realtime.
FS This is the Data Archiver computer. It acquires the data from the monitoring computers as the shots are made and copys the data to it's local C: drive. Periodically, it zips this data to files that can be restored to a local version of the TrueTrak software. In this way, data can be restored and deleted without affecting the monitoring computers.
WXXMonitoring computers are named with a leading 'W' followed by the number of the machine they are monitoring [1,99]; For machine 1 this would be W01 (W zero one). For machine 99 it would be W99.

Installation Steps


On each MONITORING computer, do the following
  1. Run "My Computer"
  2. Right click the C:\V5 directory
  3. Choose "Sharing..."
  4. Click the Shared As button and the Full Button
  5. Click "OK"

On the Data Archiver computer, do the following

The following describes installing the data archiver folder, \V5DS, on the C drive. You can install this on any drive you wish. The \V5BACKUP folder can also be anywhere you wish but if you put it on another drive you will have to edit the VISITRAK.INI file and change the setting there.

  1. Create a directory called C:\Install6.11
  2. Copy all the files from the three TrueTrak V6.11 Install/Update disks into this new directory.
  3. Run C:\Install6.11\Setup.exe 
            Directory = C:\V5DS\
        Computer  = DS
  4. Run C:\Install6.11\Setup.exe again 
            Directory = C:\V5\
        Computer  = C01
        Check This is a local installation on the Data Archiver
  5. Run My Computer
    Right click the C:\V5DS directory
    Choose Sharing...
    Click the Shared As button and the Full button
    Click OK
  6. Create a C:\V5BACKUP directory.
  7. Delete any subdirectories in the C:\V5DS\DATA\DS directory. There is one for each machine. For a new install there are three.
    C:\V5DS\DATA\DS\E20
    C:\V5DS\DATA\DS\M2
    C:\V5DS\DATA\DS\M20
  8. Edit the C:\V5DS\DATA\DS\MACHSET.TXT file and DELETE EVERYTHING in the file. Save your changes
  9. Run the NetSetup program on WS0. If you do not have a WS0 then create a C:\VisiNet folder and copy NetSetup.exe and NetSetup.chm to that folder and make a desktop icon for NetSetup.exe and run that.
  10. All of the computers should be on line. If you haven't run this program before it will ask you to pick new computers and you should pick all the True-Trak 2020 computers. Otherwise choose Add Computers... and select the new Data Server from the list of available computers. If it is not visible then you probably forgot to set the sharing on the C:\V5DS folder.
  11. Select all of the computers in the listbox and press the Standard Connections button. Then press the Save Changes button.
  12. Restart the True-Trak 2020 programs, including the Board Monitor, on the montoring computers.
  13. Run the "Data Server Backup" program by double-clicking the icon on the desktop.
  14. Pull down the Edit menu and choose Shot Save Configuration. Press Save Changes.
  15. Pull down the Edit menu and choose Global File Settings. Press Save Changes.
  16. You should soon see the shots begin to come in from the one monitoring computer you added the "DataServer=Fs" line to.
  17. Pull down the Edit menu and choose Global File Settings and press the Save Changes button
  18. Set up the rest of the monitoring computers, one at a time. If there is a problem, shut down the "Data Server Backup" program and redo steps 8 and 9, deleting any machines that exist (including their directories).
    You should then be able to do steps 10 and 11 again to confirm that you are able to talk to the computer.
  19. Edit the C:\V5\EXES\PROFILE.MNU file with the text editor.
    Change the number on the end of the following lines to 15 
        1,FasTrak &Board Monitor,703,9
    2,SEPARATOR,-1,9
    1,Sure&Trak Control Interface,710,4
  20. Copy the DsBackup.exe, DsBackup.ini, and DsBackup.chm files from the Data Server Install disk to the new \V5DS\Exes folder and make a desktop icon for the DsBackup.exe program.
  21. Copy the DsRestore.exe program to the C:\V5\Exes folder and make a desktop icon.
  22. Edit the C:\V5\EXES\VISITRAK.INI and add a line in the [Config] section that says:
    NoNetwork=Y

Data Server Computer Notes

  • Control Panel/Network
    • Configuration
      • Client for Microsoft Networks
        Dlink
        NetBEUI
        File and printer sharing for Microsoft Networks
    • Identification
      • Computer Name: FS
        WorkGroup: VISITRAK
    • Access Control
      • x Share-level access control
  • C:\V5
    • Standalone installation One computer (C01) Local copy of each machine.
  • C:\V5BACKUP
    • Directory for backup zip files
  • C:\V5DS
    • Data Server network installation
      This directory must be fully shared.
      The C:\V5DS\DATA\COMPUTER.TXT file should contain a line for every computer except WS0.
      This Computer = DS
  • DSBACKUP.EXE
    • This is the backup program. It should be in the C:\V5DS\EXES directory.
      There should be an icon for this.
  • DSRESTOR.EXE
    • This is the network restore program. It should be in the C:\V5\EXES directory.
      There should be an icon for this.

Office Workstation (WS0) Computer Notes

There should be no machines at this computer (see steps 8 and 9 for how to delete the machines).
