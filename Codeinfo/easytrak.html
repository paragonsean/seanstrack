EasyTrak

The EasyTrak screen is shown below. The Current Position is the position of the rod.

The Velocity Averages each represent the average velocity over the range below the value. For example, in the screen below the first range is from 0 to .05 inches. The average velocity in this range during the last shot was 5.47 ips.

The shot is controled in the region under the Control Settings.

On the left are the positions and percent valve openings for each position. The first position is always 0.00.

To the right of these are the four Limit Switch Positions. These switches will be closed when the current position meets or exceeds the corresponding position and will be opened when the current position is less than the corresponding position.

To the right of this is the Tie Bar Monitor which shows the current tonnage of each strain gauge and the total.

You can use the arrow keys to move from one field to another. When you press the enter key the cursor is automatically moved to the next field.

Note: Any change to a field is immediately sent to the EasyTrak board. Because of this you should not make changes during a shot.

Pressing the <F2> key saves the current settings to disk. This is also done when the program exits. These settings are read in when the program starts.

The SENSORS.TXT file tells the program what hardware is being used. The first entry in the file describes the settings for the position sensor.

This is followed by four listings, one for each of the sensors connected to the four analog inputs.

Finally there is a setting for the minimum output voltage desired and the voltage range represented by a 100% valve open. The default settings for this are shown below.

;Min voltage [0, 10)
0.0
;Voltage Range (0, 10-min];
10.0

If you need an output voltage range of 1 to 5 volts you would use the following settings.

;Min voltage [0, 10)
1.0
;Voltage Range (0, 10-min];
4.0
