Installing the TrueTrak 2020 Monitoring Software for the FasTrak2

Copying the software from the CD to the Data Collection Terminal (DCT)

  1. Run the V604.exe program. This will create a C:\Install6.04 folder on your computer.
  2. Run the FT2_Install.exe program. This will create a C:\Ft2_Setup folder on your computer.
  3. Open the C:\Install6.04 folder and run the Setup.exe program. This will install the TrueTrak 2020 ™ monitoring software on your system.
  4. On the opening screen, choose Install New TrueTrak System
  5. On the next screen click the checkboxes next to
    • Start Part Editor at TrueTrak Startup
    • Start Board Monitor at TrueTrak Startup
    • FasTrak2
  6. Press the Continue Button to complete the installation.

Setting up the TCP/IP Addresses

  1. Determine the TCP/IP addresses that you are going to use for the computer and the Board. If you are going to have more than one FasTrak2 board on the same network you will need a separate fixed address for each board. The default address for the board is 192.168.254.30 with a subnet mask of 255.255.255.0.
  2. If you are using one of our computers, the computer will probably be set up with an address of 192.168.254.40.
  3. If you need to change the address of the FasTrak2 board you will need to connect a serial cable from the COM1 port of the computer to the serial port of the FasTrak2 board.
  4. You will then need to put the FasTrak2 board into Germs mode by holding down the SW7 button, pressing and releasing the SW9 button, and then releasing the SW7. If you have done this correctly you will see that the led displays will show 88.
  5. You will then need to open an MS-Dos window. This is called a Command Prompt on newer versions of Windows.
    Once in the MS_Dos window, enter CD \Ft2_Setup to move to that folder and then enter T to upload the program that allows you to change the TCP/IP address of the board.
  6. Once the program has been uploaded you will be shown the current network address. At the prompt, press the A key to go to the Network Settings menu and then the C key to change the settings.
  7. The first address that is displayed is the hardware address of the board. This does not need to be changed unless you have more than one FasTrak2 board on the same network. In that case you should change the last number of the hardware address so that each FasTrak2 board has a different one.
  8. The next address is the TCP/IP address. Change this as necessary.
  9. Skip the nameserver ip address by pressing Enter.
  10. The subnet mask is displayed next. Change this if you need to.
  11. Skip the gateway ip address.
  12. Press B to save the changes and then, when this is completed, press Q to return to the main menu and then C to exit the program.
  13. Press the SW9 button on the FasTrak2 board to reset the board.

Setting up the Monitoring Software

  1. Double click the TrueTrak 2020 Monitoring System icon that has been put on the desktop. This will run the programs.
  2. Pull down the Monitor menu on the main screen and choose FasTrak Board Setup.
  3. Change the Board Type of board number 1 from Single to FasTrak2.
    4. b
  4. Enter the TCP/IP address of the board into the Address editbox and 20000 into the Port editbox.
  5. Press Save Changes
  6. Bring up the Part Setup program and choose the part you are going to monitor. Make any changes necessary on the part setup and then press the Save Changes button to save the setup. Press this even if you have not made any changes.
  7. Pull down the Monitor menu and choose Monitor Setup and make sure the correct machine and part are selected to be monitored.
  8. Bring up the FasTrak2 monitoring program and exit from the program. Then rerun the program by pulling down the Monitor menu and choosing FasTrak Board Monitor. This time the program should show Connected in the Connected column.
