Insert the floppy disk you wish to use into the floppy drive.

Open "MyComputer" and click once on the Floppy drive.

Pull down the "File" menu and click on the "Format" selection as shown below.

When the format window appears, set the "Format type" to "Full", check the "Copy system files" option at the bottom, and then press the "Start" button.

When the format is complete the Format Results screen should show "0 bytes in bad sectors" as shown below. If the number of bad sectors is not zero you should throw the floppy away and repeat the format with another floppy.

With the floppy disk in the drive start the "Part Setup" program and press the "Shot Control" tab. Press the "MS-DOS" button at the bottom left of the screen.

Press the "Begin" button on the "Backup to MSDOS Boot Disk" screen to begin the backup. This window will disappear when the backup is complete.
