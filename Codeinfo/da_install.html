Data Archiver Installation Steps

  • Go to http://www.visi-trakworldwide.com/get.htm and get the v649.exe file.
  • Copy this to the root directory of the C drive on the Data Archiver and run it to create a C:\Install6.49 directory. If that directory already exists, just tell the program to replace always.
  • Run the Setup.exe program in that directory
  • Press the Install New TrueTrak System button. A screen with options will appear.
  • Change the directory from C:\V5\ to C:\V5DS\
  • Change the name of the computer from C01 to DS. This is only a local name used by the programs and will not affect the actual network name of the computer.
  • At the bottom of the screen check the checkbox for This is an Office Workstation or Data Archiver.
  • Press the Continue button to install the software
  • You will have created a new C:\V5DS directory. Make this Shared Full
    • Right click on the V5 directory
    • From the popup menu choose Sharing and Security...
    • Check Share this folder on the network
    • Check Allow network users to change my files
  • Run the C:\Install6.49\Setup.exe program again.
  • Press the Install New TrueTrak System button.
  • At the bottom of the screen check the checkbox for This is a local installation on the Data Archiver.
  • Press the Continue button to install the local copy of the software.
  • Go to http://www.visi-trakworldwide.com/getda.htm and get the da.exe file.
  • Copy this file to the root directory of the Data Archiver and run it from there to create the C:\Install.ds directory.
  • Run the C:\Install.ds\Setup.exe program
  • Press the Begin button to install the software.
  • Press the Esc key to exit from the install program when it is finished.
  • You should have a new icon on the desktop for the Data Archiver Backup Program. Double click this to run the program.
  • Pull down the Edit menu and choose Global File Settings and then press the Save Changes button.
  • Go to http://www.visi-trakworldwide.com/getnet.htm and get the ns.exe file.
  • Copy this to the root directory of whichever computer currently has the NetSetup program on it. Normally this is the Office Workstation but it might be the Data Archiver. The following directions will assume it is on the Office Workstation.
  • Run the ns.exe from the root directory to create the C:\Install.ns directory on the Office Workstation.
  • Run the C:\Install.ns\Setup.exe program.
  • Press the Begin button to install the software.
  • Press the Esc key to exit from the install program when it is finished.
  • Make sure all the TrueTrak computers are online.
  • You should have a new icon on the desktop for the NetSetup program. Double click this to run the NetSetup program
  • If there is a current entry for the Data Archiver, remove it
  • Press the Browse for Computers button and browse to the Data Archiver and click once on the V5DS directory to add it to the list of computers.
  • If there are any other TrueTrak computers that are not listed in the listbox, browse over to each of these and click once on the V5 directory found there. Make sure the type of computer is set correctly for each one that you add.
  • Select all the computers in the main listbox (click the first and then shift-click the last).
  • Press the Standard Connections button
  • Press the Save Changes button
