How to Install the Konica Network Printer

  1. Press Start in the lower left corner
  2. Choose Settings
  3. Choose Printers
  4. Double Click Add Printer
  5. Network Printer
  6. Press Brouse
  7. Brouse to \\visitrak\visitrak_org
  8. Choose KCA_031621-P1
  9. Press Have Disk
  10. Brouse to F:\Users\Common\Konica\win95_setup and press Ok
  11. Select Konica 7135 PS
  12. The Konica install program will now start up. When the property sheet appears click on the Postscript tab and then press the Postscript Options button. Under Send Ctrl-D to Printer select Ctrl-D at Start and End
  13. Check the Use Postscript Pass-through checkbox.
  14. Press Ok.
  15. Click on the Device Options tab. Choose the following.
  16. Finisher FS-107 (2-trays)
  17. Fax Unit Yes
  18. Base Unit DB-209/210
  19. ADF Yes
  20. 64MB
  21. Press Apply and then Ok
  22. A new icon appears on the Printers Folder, right click and choose Properties
  23. Select the General tab and print a test page

The install for win2000 is almost the same except you go to the win200_setup folder. The program doesn't let you set up all the options initially but you can go to properties later and do this.
