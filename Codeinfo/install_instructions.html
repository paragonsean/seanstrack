You can download the current english version of the software here

http://www.visi-trakworldwide.com/get.htm

The program you will download will be v639.exe. Copy this to your hard drive and run it. It will create a C:\Install6.39 directory and copy the installation files there.

Now go to the following page

http://www.visi-trakworldwide.com/getnet.htm

and download ns.exe. Copy this file to your hard drive and run it. I will create a C:\Install.ns directory and copy the installation files for the NetSetup program there.

I am assuming you are installing a new system. On each DCT do the following.

  • Run C:\Install6.39\Setup.exe
  • Press the Install New TrueTrak System button.
  • Check the box next to Start Board Monitor at TrueTrak Startup.
  • Check the box next to Automatically Start when the Computer Boots.
  • Press the Continue button to begin the installation
At the office workstation, do the following.
  • Run C:\Install6.39\Setup.exe
  • Press the Install New TrueTrak System button.
  • Check the box next to This is an Office Workstation or Data Archiver.
  • Press the Continue button to begin the installation
  • When this is done, close the installation program and run C:\Install.ns\Setup.exe. Press the Begin button to install the NetSetup program. When it is done, press the Esc key to close the install program.

At each DCT, do the following.

  • Run the True-Trak Monitoring Program.
  • Pull down the Monitor menu and choose Machine Setup.
  • Click on the machine that is most like yours. E20 uses english units (inches and IPS) and has a 20 pitch rod. M20 uses metric units (mm and mps) and has a 20 pitch rod. M2 uses metric units and has a 2 mm rod.
  • Pull down the Edit menu and choose Create.
  • Give the new machine a unique name and press OK
  • Pull down the Monitor menu and choose Monitor Setups
  • Click on the new machine and click on "Monitor This Part".
  • Click on Save Changes.
  • Close the Machine Setup program and start it again.
  • Click on each of the original machines (E20,M20,M2) and delete them by pulling down the Edit menu and choosing Delete.
  • Set up network file sharing and make the C:/V5 directory shared with full access.
  • Make the network names of the DCTs "W01" and "W02".
At the Office Workstation
  • Set up the network file sharing and make the C:/V5 directory shared with full access.
  • Make the network name of this computer "WS0"
  • Run the C:\Visinet\netsetup.exe program.
  • Click on the Brouse for Computers button and brouse to each of the three computers as instructed on the top of the window. Click ONCE on each V5 folder icon to add that computer to the listbox.
  • Highlight all of the computers.
  • Click Standard Connections.
  • Click Save Changes.
