You will need four blank, formatted, diskettes. Label them

  • Disk 1
  • Disk 2
  • Disk 3
  • Update 4.32

I am assuming that you are in the office, working on a Windows 95, 98, NT, etc., computer and that the four files, Disk1.exe, Disk2.exe, Disk3.exe, and Update4.exe, have been downloaded onto the hard drive of that computer.

Put the first diskette, Disk 1, into the diskette drive and run the Disk1.exe program. This will copy the necessary files to Disk 1.

Repeat this with
Diskette NameProgram to Run
Disk 2Disk2.exe
Disk 3Disk3.exe
Update 4.32Update4.exe

Now that you have created the installation and update diskettes, I assume you are doing the rest of the installation on the MS-DOS based monitoring computer.

Full Installation

Diskettes 1 through 3 are used to do a full installation of the program. This will delete all existing data on the computer. If you are going to do this you should first remove all Visi-Trak directories (you can back up your current system with backupv4.exe first if you wish). Delete the following directories if they exist.

  • V4
  • V4LOCAL
  • PDOXRUN
  • PRIVATE

For example, to delete the V4 directory enter the following two lines.
CD \
DELTREE V4

Now, insert Disk 3 into the diskette drive and enter the following two lines.
A:
INSTALL A C

You will be prompted to insert each of the three diskettes. This will install a Version 4.29 of the software. When the installation is done, continue with the update to Version 4.32 as described below.

Update an Existing installation

Insert the Update 4.32 diskette into the diskette drive and enter the following two lines.
A:
UPDATE A C

This will update your system to the current version. You can start the program by entering

V4
