Current Monitor Part

The Current Monitor Part is the part currently selected to be monitored. You choose this in the Monitor Setup program.

Source Folder

The program simulates monitoring by copying shots, one by one, from a source folder which you can choose.

To choose a new source folder press the Brouse button and brouse to a folder that contains shots, which look like 003012.pro. Select one of the shots, it doesn't matter which one you select as they will all be used, and then press the Open button to select this folder as the current source. The folder path will be shown in red when you first select it. When you press the Start button the program will begin using the new source folder and will change the color of the path to black. Cycle Time

This is used to tell how often to make a shot if the Manual Mode checkbox is not checked.

Manual Mode

If you check this a small cycle start (CS) button will appear in the upper right hand section of the screen. Pressing this button will make a shot.

Continuous

If this is checked then when the last shot file has been read the program will start again at the start of the list. If this is not checked then the program will stop making shots after reading all of the profiles.

Start

Pressing this button tells the program to begin getting shots from the displayed source folder. If the path name of the source folder is shown in red then it will be changed to black once you have pressed this button.

In any case, the program makes a list of all the profiles in the selected source folder, sorts the list, and then begins getting shots from the folder beginning with the first entry in the sorted list of profiles.
