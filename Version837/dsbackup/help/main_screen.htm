Main Screen

The main screen of the program is shown here.

The Status line at the bottom shows the most recent shot notification. The listbox shows the number of shots received from each machine that is being monitored. These are the number that have not been saved. When the shots are zipped up the count for that machine is reset to zero.

Normally you set the shots to be zipped either after a certain number (100 in the picture here) have been received from a machine or else at a certain time of the day. Up to three times can be specified.

You can also press the Zip Now button at any time to zip all of the current data.

There are two choices under the Edit menu that allow you to make changes to the way the program saves.
