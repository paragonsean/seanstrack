Main Screen

Board

This column shows the number of the board. This is the same number that is shown in the FasTrak Board Setup program. There is a radio button next to the number which, if you have more than one FasTrak2 boards, allows you to select which board the diagnostic programs will talk to.

Address and Port

These columns show the current TCP address and port number used by the boards. These are set in the FasTrak Board Setup program.

Connected

This column shows Connecting when the program tries to contact the board. Normally this will change almost immediately to Connected when the first response is received. If no response is received Not Connected is displayed. If Not Connected is displayed, the program will check every minute or so to see if the board is present and will display Connecting followed by either Connected or Not Connected. If a board is not set up to be monitored then Not Monitored will be displayed and the program will not attempt to contact the board.

Machine and Part

This is the machine that the FasTrak2™ board is monitoring and the current part.

Monitor Status

The monitor status is blank when the program first connects to the board. It will show  Cycle Start  until the shot data has been received and then Ready when the board is ready to receive another shot.

New Setup is displayed whenever the program has downloaded a new part setup to the board. This remains displayed until the next cycle start is received.
