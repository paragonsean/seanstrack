Uploading Flash Files Through the Serial Port

Before upload the flash files to the FasTrak2™ board you must put the board into GERMS mode.

To put the FasTrak2™ board into GERMS mode press and hold SW6. Then, while still pressing SW6, press and release the RESET button and then release SW6.


You must upload the files in an MS-DOS window. Press the Start button in the lower left corner of the screen. Click on Programs, and then on MS-DOS Prompt. This will open an MS-DOS window. Set your current folder by entering the following command. 
cd  \ftii
 Begin uploading the first file by entering the following command. 
f1
 You will see 
Fastrak2: Downloading.........
 The program will continue to put out dots until it is finished uploading. Then you will see the following line 
FasTrak2: Terminal mode (Control-C exits)
 Hold down the Ctrl key and press the C key at the same time. This will exit back to the command line. Now repeat the upload procedure by entering 
f2
 When this file has completed, exit to the command line again and enter 
f3
 When this file has completed you are finished uploading files.
You must now RESET the FasTrak2™ board. You do this by simply pressing the RESET button on the FasTrak2™ board (the RESET button is the same one you pressed along with SW6 to get the board into GERMS mode).
