File Name

This is the path to the file that will be created for the shot parameter records.

There are several tags that can be used to insert dynamic content into the file names. These are as follows.

TagContent
{s} Shift Number
{dd} Day of the month [1,31]
{mm} Month [1,12]
{yyyy} Year
{m} Machine
{c} Computer
{p} Part

Thus the following entry

FileName=C:\V5NetFiles\{mm}\{dd}\Shotparm.csv

would create a new file each day, beginning with

  • C:\V5NetFiles\05\01\Shotparm.csv
  • C:\V5NetFiles\05\02\Shotparm.csv
  • ...
  • C:\V5NetFiles\05\31\Shotparm.csv
The day of the month in this case would be the actual day of the month on which the shot was taken. Note that if you include the shift number in the file name then the date that is used is the shift date, which is not necessarily the same as the shot date.
